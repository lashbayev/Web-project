# Web-Project


## Project description
This project is a web application designed to help students find internships, pet-projects, and teams to collaborate with.

The application will allow users to:

- register and create a personal profile (skills, GitHub link, experience level)
- browse a feed of available internships and open pet-projects
- filter opportunities by programming language, skill level, or project type
- apply and respond to projects or internship listings
- connect with other students and form teams

The goal of the project is to provide a simple and accessible platform that helps students gain real experience, grow their skills, and find like-minded peers.
